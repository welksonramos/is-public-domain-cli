# is-public-domain-cli
>  Check if a domain is public

# Install

```bash
$npm install --global is-public-domain-cli
```

# Usage

```bash
domain --help

Check if a domain is public

Usage:
$ domain <url>
Example
domain sindresorhus.com

```
